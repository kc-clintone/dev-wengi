# Level 1 — DOM & Interactivity

Welcome to Level 1 🚀

This is where your code starts to **do things**.

You’ll learn how to connect JavaScript to your HTML and make your pages interactive.

---

## 🎯 Goal

By the end of this level, you should be able to:

- Select elements from the page
- Respond to user actions (clicks, typing)
- Update the UI dynamically
- Build a small interactive app

---

## 🧠 What You’ll Learn

- DOM selection (`querySelector`)
- Event handling (`click`, `input`)
- Updating content (`textContent`, `innerHTML`)
- Basic state (keeping track of values)

---

## 📚 Tasks

1. Select elements from the DOM
2. Handle user events
3. Update the UI dynamically
4. Build a mini project
