# Task 4 — Mini Project (Counter App)

## 🎯 Goal

Build your first interactive app.

---

## 🛠 Requirements

Create a counter app with:

- A number displayed
- A "+" button
- A "-" button

---

## Behavior

- Clicking "+" increases the number
- Clicking "-" decreases the number

---

## 💡 Hints

- Keep track of a number (state)
- Update UI after every click

---

## 🧠 Challenge (Optional)

- Prevent number from going below 0
- Add a reset button

---

## ✅ Done When

- Counter updates correctly
- UI reflects changes instantly
