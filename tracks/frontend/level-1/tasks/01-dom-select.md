# Task 1 — Select Elements

## 🎯 Goal

Learn how to access elements from your HTML using JavaScript.

---

## 🛠 Requirements

- Create an HTML page with:
  - A heading
  - A button

- Use JavaScript to:
  - Select the heading
  - Select the button

---

## 💡 Hints

- `document.querySelector()`
- Use class or id

---

## ✅ Done When

- You can log elements in the console
