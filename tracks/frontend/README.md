# Frontend Track

Build real user interfaces from scratch and grow into a confident frontend developer.

This track is **hands-on**. You won’t just learn concepts — you’ll use them to build things.

---

## 🚀 How This Track Works

1. Start at **Level 0**
2. Complete all tasks in order
3. Submit your work
4. Move to the next level

Each level increases in difficulty and moves you closer to working on real projects.

---

## 🧭 Levels

- **Level 0 — Foundations**  
  HTML, CSS, basic JavaScript, environment setup

- **Level 1 — DOM & Interactivity**  
  Events, DOM manipulation, simple apps

- **Level 2 — Modern Frontend**  
  React (or similar), components, state

- **Level 3 — Real Project**  
  Work on an actual project from `/projects`

---

## 🎯 Goal

By the end of this track, you should be able to:

- Build responsive web pages
- Write clean, readable JavaScript
- Understand how the browser works
- Contribute to real frontend projects

---

## ⚠️ Rules

- Don’t skip levels (unless you know what you're doing)
- Don’t copy-paste blindly
- If you’re stuck, **struggle first**, then research
- Ask any questions on the community platforms (You can share anything as well)

---

## 🛠 What You’ll Need

- A code editor (VS Code recommended)
- A browser (Chrome/Firefox)
- Git + GitHub account
