# Level 0 — Foundations

Welcome to Level 0 👋

This is where you build your foundation in:

- HTML
- CSS
- JavaScript
- Developer workflow

---

## 🎯 Goal

By the end of this level, you should be able to:

- Create a basic web page
- Style it with CSS
- Add simple interactivity with JavaScript
- Use Git to track your work

---

## 📚 Tasks

Complete these in order:

1. Setup your environment
2. Create your first HTML page
3. Style it with CSS
4. Add JavaScript
5. Push your project to GitHub

---

## 🧠 How to Approach This

- Try first before searching
- Break problems into small steps
- Use Google when stuck (this is part of the skill)
- Don't forget our community platforms (Discord & Github Discussions)

---

## ✅ Completion

You’re done when:

- All tasks are complete
- Your code is pushed to GitHub
- You understand what you built
