# Task 5 — Push to GitHub

## 🎯 Goal

Learn basic Git workflow.

---

## 🛠 Requirements

- Initialize git
- Commit your code
- Push to GitHub

---

## 💡 Commands

```bash
git init
git add .
git commit -m "level 0 complete"
git branch -M main
git remote add origin <your-repo-url>
git push -u origin main
```

## ✅ Done When

- Your project is on github
