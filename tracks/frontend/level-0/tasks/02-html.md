# Task 2 — Build a Basic HTML Page

## 🎯 Goal

Understand structure of a web page.

---

## 🛠 Requirements

Create an `index.html` file with:

- A title
- A heading
- A paragraph
- A button

---

## 💡 Hints

- Use `<!DOCTYPE html>`
- Use `<h1>`, `<p>`, `<button>`

---

## ✅ Done When

- Your page opens in a browser
- You see your content rendered
