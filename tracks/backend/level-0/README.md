# Level 0 — Backend Foundations

Goal: Understand basic backend concepts, APIs, and server setup.

## Tasks
1. Set up your backend environment (Node.js / Python / etc.)
2. Create a simple server
3. Define basic routes (e.g., /, /hello)
4. Return JSON responses
5. Connect to a simple data source (in-memory or file)

## How to Complete
- Fork this repo
- Complete tasks inside /tasks
- Run your server locally to verify endpoints
- Submit a PR

Once approved → move to Level 1 🚀
